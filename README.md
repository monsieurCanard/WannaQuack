# Wannacry - Histoire

L'attaque du ransomware WannaCry, également connu sous le nom de WanaDecrypt0r 2.0, a commencé à proliférer à travers l'Europe le 12 mai 2017. Cette attaque est l'une des plus importantes jamais enregistrées et s'est propagée dans plus de 140 pays et utilise un exploit appelé **ETERNALBLUE**[^1] pour se propager rapidement entre les ordinateurs vulnérables.

## Références

[^1]: **ETERNALBLUE** : Exploit développé par la NSA et divulgué par le groupe Shadow Brokers en avril 2017. Il exploite une vulnérabilité critique dans le protocole SMB (SMBv1) de Windows, permettant l'exécution de code à distance sans authentification.
